はじめに

INXPO は、テレビスタイルのイベント体験を作成するためのビデオストリーミングプラットフォームです。より多くのオーディエンスにリーチするには、Brightcoveライブを活用してイベントをストリーミングできます。

ステップ

Brightcoveライブでミーティングを主催するには、次の手順に従います。

  1. Studio で、{% if site.product_short == "Video Cloud" or site.product_short == "Beacon" or site.product_short == "CorpTV" %} を作成します。 Live モジュールを使用したライブ イベント. {% endif %} {% if site.product_short == "Live" %} Live モジュールを使用したライブ イベント. {% endif %}
    完了後にブロードキャストを保存する場合は、必ず完了時にイベントをビデオ アセットに変換するチェックボックス！
  2. ライブ イベントを作成したら、ストリーミング RTMP URL とストリーミング キーを制御室パネル。
    • ストリーミング RTMP URL -ストリーミング エンドポイント URL (エンドポイント URL は、Brightcove Live Control Room で見つけることができます。
    • ストリーミング キー-ストリーム名(ブライトコーブでは、ストリーム名は常に"生きている" )
  3. Webサイト/イントラネットページに、このライブイベント用のプレーヤーを公開して埋め込み、ページのURLを書き留めます。

ストリーミングセッションの設定

ストリーミング RTMP URL とストリーミングキーを使用して、INXPO プロデューサーツールで設定する必要があります。

  1. INXPO プロデューサーツールに移動し、3 つのドットアイコンをクリックして [出力詳細] セクションに移動します。
  2. 鉛筆アイコンをクリックします。
  3. 以下の詳細に、Brightcoveのライブイベントの認証情報を入力します。
  4. の中に題名フィールドに、Brightcove ライブ ストリーム名を追加します。
  5. の中にエンコーディング プロファイルフィールドで、エンコーダーが Web キャストを圧縮して出席者に配信する方法を定義するデフォルトのエンコード プロファイルを選択します。
  6. の中にRTMP URLフィールドに、Brightcove ストリーミング エンドポイント URL を挿入します。
  7. クリックわかったボタン。
