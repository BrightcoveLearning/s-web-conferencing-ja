はじめに

Cisco Webex オンライン他の人と会うことができるビデオ会議アプリケーションです。より多くのオーディエンスにリーチするために、Brightcoveライブを活用してミーティングをストリーミングして、他のユーザーが見られるようにすることができます。

ステップ

Brightcoveライブでミーティングを主催するには、次の手順に従います。

  1. サイト管理者として Cisco Webex プラットフォームにログインします。
  2. 次のいずれかを使用してサイト管理またコントロール ハブ、 有効ライブ ストリームへのアクセスにその他のストリーミング サービス.
  3. 次のいずれかを使用してサイト管理またコントロール ハブ、個々のユーザーのライブ ストリーミングを有効にします。

    案内するユーザー更新するユーザー アカウントを選択します。クリック高度なユーザー設定とトラッキング コードユーザーのセクションを確認し、ライブ ストリーミングを有効にする箱。

  4. Studioにログインします。
  5. Live モジュールを開き、ライブ イベントを作成します。 {% if site.product_short == "Video Cloud" or site.product_short == "Beacon" or site.product_short == "CorpTV" %} 詳細については、 Live モジュールを使用したライブ イベントの作成と管理. {% endif %} {% if site.product_short == "ライブ" %} 詳細については、 Live モジュールを使用したライブ イベントの作成と管理. {% endif %}
    ノート：完了後にブロードキャストを保存する場合は、必ず完了時にイベントをビデオ アセットに変換するチェックボックス。
  6. このイベントを作成すると、次の情報が提供されます。
    • エンドポイント (RTMP URL) - ストリーミング URL
    • ストリーム名- ストリーミング キー。値は常に生きている
  7. このライブイベントのプレーヤーを公開し、ウェブサイトまたはイントラネットページに埋め込みます。ページ URL をクリップボードにコピーします。
  8. Cisco Webex アプリケーションに戻ります。Webex ミーティングを開始またはスケジュールします。これを行うには、縦に並んだ 3 つのドットのアイコンを選択し、ライブストリーミングを開始.
  9. の中にその他のストリーミングサービス情報ダイアログで、次のように入力します。
    • ストリーミングサービス:使用するブライトコーブ
    • ターゲット ストリーム リンク:使用エンドポイント (RTMP URL) Brightcove Live からの値。
    • ターゲット ストリーム キー:使用する生きている(常にこの値です)
  10. それだおまえは生きてる！
