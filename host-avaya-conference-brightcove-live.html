はじめに

AvayaRoomsSystemsは、オンラインで他の人と会うことができるビデオ会議アプリケーションです。より多くのオーディエンスにリーチするために、Brightcoveライブを活用してミーティングをストリーミングして、他のユーザーが見られるようにすることができます。

ステップ

Brightcoveライブでミーティングを主催するには、次の手順に従います。

    {% if site.product_short=="VideoCloud"またはsite.product_short=="Beacon"またはsite.product_short=="CorpTV"%}
  1. Studio で、 Live モジュールを使用したライブ イベント. {% endif %} {% if site.product_short == "Live" %}
  2. Studio で、 Live モジュールを使用したライブ イベント. {% endif %}
    ノート：完了後にブロードキャストを保存する場合は、必ず完了時にイベントをビデオ アセットに変換するチェックボックス。
  3. ライブ イベントを作成したら、ストリーミング RTMP URL とストリーミング キーを制御室パネル。
    • ストリーミング RTMP URL -ストリーミング エンドポイント URL (エンドポイント URL は、Brightcove Live Control Room で見つけることができます。
    • ストリーミング キー -ストリーム名(ブライトコーブでは、ストリーム名は常に"生きている" )
  4. Web サイト/イントラネットページにこのライブイベントのプレーヤーを公開して埋め込み、ページの URL を書き留めます。

ストリーミングセッションの設定

ストリーミング RTMP URL とストリーミングキーを使用して、Avaya コーデックユニットで設定する必要があります。

  1. Avaya GUI で、設定 > ストリーミング.

    (オプション) Avaya Web サイトで、管理者設定 > ユーティリティ > ストリーミング.

  2. ストリーミング RTMP URL を挿入します。
  3. ストリーミングキーを挿入します。

ストリーミングセッションの開始/停止

ストリーミングセッションを開始または停止するには、次の手順を実行します。

  1. に行くホームページ > もっと > ストリーミング.
  2. クリックストリーミングボタン。これにより、ライブストリームが開始されます。
  3. RTMP ストリーミングはデフォルトで有効になっています。無効にするには、Avaya プラットフォームで構成 > 詳細 > ユーティリティ > ストリーミング.
